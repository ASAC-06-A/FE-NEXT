## 프로젝트 개요

StudyHub은 사용자가 강의 사이트 주소를 북마크하고 자신만의 강의 컬렉션을 생성할 수 있도록 돕는 플랫폼입니다.  
또한 학습 로드맵을 체계적으로 설계하고 관리할 수 있는 기능을 제공합니다.
